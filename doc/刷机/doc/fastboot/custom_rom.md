 # Fastboot刷写第三方ROM

刷机之前，建议简单了解下刷机原理  
特别是[分区](https://segmentfault.com/a/1190000021601415)的概念  
从原理来讲，刷第三方ROM和系统更新并没有本质区别

( 时效性警告！ ) 目前的索尼设备有这几种分区类型 

## A-Only 分区
骁龙835处理器平台(即XZ1/XZP/XZ1c)以及之前的所有设备 

## 无动态分区的A/B设备

(包括)Xperia XZ2系列到Xperia 5之间的所有设备



## 动态分区的A/B设备

Xperia 1, 5, 10 的 II 和 III 代
